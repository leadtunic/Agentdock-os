# Code of Conduct

AgentDock OS is a professional open source community for building safer AI agent infrastructure.

## Expected behavior

- Be respectful.
- Be constructive.
- Assume good intent.
- Give actionable feedback.
- Keep discussions focused on the project.

## Unacceptable behavior

- Harassment or personal attacks.
- Discriminatory language.
- Threats or intimidation.
- Publishing private information.
- Spam or bad-faith participation.
