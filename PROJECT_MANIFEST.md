# Project Manifest

This zip contains a complete planned AgentDock OS repository foundation based on the product specification.

Included:

- Complete `SPEC.md`
- README and community files
- GitHub templates and CI workflows
- Docker Compose and Dokploy Compose
- Next.js dashboard skeleton
- FastAPI API skeleton with core models/routes
- Agent runtime service skeleton
- Browser runtime service skeleton
- Gateway service skeleton
- Worker skeleton
- Shared packages and toolbar package
- Plugin skeletons
- Documentation tree
- Infra references

This is intentionally structured as the full product architecture from the beginning, not a single-screen MVP.
